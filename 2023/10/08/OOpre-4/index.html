import java.util.Scanner;
import java.util.regex.Matcher;
import java.util.regex.Pattern;

public class Regex{
public void example(){
Scanner scanner = new(Sysetem.in);
String PATTERN = “(\d{1,4})/((1[0-2])|(0?[1-9]))/(([1-2][0-9]|30|31)|(0?[1-9])) ((1[0-9])|(2[0-3])|(0?[0-9])):([0-5]?[0-9]:?){2} \w+”;

  /*创建要匹配的模式*/  
  Pattern pattern = Pattern.compile(PATTERN);
  /*java中的不定行输入*/
  while(scanner.hasNextLine()){
    String str = scanner.nextLine();
    System.out.println(str);
    /*创建一个匹配“对象器” */
    Matcher matcher = pattern.matcher(str);
    /*如果匹配成功*/
    if(matcher.find()){
      /*将匹配到的字符串进行分割*/
      for(String s: matcher.group().split(" ")){
        System.out.println(s);
      }
    }
    /* 如果匹配失败 */
    else{
      System.out.println("not found");
    }
    System.out.println("==========");
  }
}

}

(hello world)* –> 空串/hello world/hello worldhello world……
(hel*o world) –> heo world/helo world/hello world……

(hel+o world) –> helo world/hello world……

(hello{3}) –> hellooo
(hello{1,}) –> hello,helloo……
(hello{1,3}) –> hello,helloo,hellooo

(\d{1,4})/((1[0-2])|(0?[1-9]))/(([1-2][0-9]|30|31)|(0?[1-9])) ((1[0-9])|(2[0-3])|(0?[0-9])):([0-5]?[0-9]:?){2} \w+ 


+ 第一个括号，表示对任意数字匹配1-4次,也即可以匹配1-4位数的年份，从0-9999年均可。
+ 中间的斜杠表示匹配斜杠，后同
+ 第二个括号，表示月份，第一个小括号表示10，11，12月，第二个小括号表示月份1-9，且有无前导0均可，两个小括号之间再用|连接。
+ 第三个括号，表示一个月中的几号，第一个小括号表示双位日期10-31，第二个小括号表示单位日期1-9，且有无前导0均可，两个小括号再用|连接。注意第三四个括号之间存在空格需要匹配
+ 第四个括号，表示小时,翻译过来即(10-19)|(20-23)|(0-9)，有无前导0均可
+ 第五个括号以及后方的`{2}`表示分钟和秒数第五个括号翻译过来即为0-59，最后的冒号可匹配也可不匹配（分钟后需要冒号而秒数后不需要冒号），再重复两遍，即得分钟和秒数。
+ 最后的`\\w+`，表示至少有一个字符，且该字符不是`\n`。

### 分组匹配
+ 用()把要提取的规则分组
+ 按括号提取分组后的子串，group(1)等，group(0)提取输入的整个字符串  
### 非贪婪匹配
正则匹配默认使用贪婪匹配，任何一个规则，他总是尽可能多的向后匹配。  
**注意找到特征字符串以区分特征**
