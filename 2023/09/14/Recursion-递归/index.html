Recursion(递归)

递归，也就是方法自己调用自己。

递归重要规则

  • 每执行一次方法，就创建一个新的受保护的独立空间，称为栈空间。
  • 方法中的局部变量独立，不会相互影响（基本数据类型），如果是引用类型变量，例如数组、对象等，就会共享变量的数据。
  • 注意！递归必须要设置退出递归的条件，且要向该条件逼近，否则会造成无限递归，栈溢出(StackOverflowError)
  • 方法执行完毕，或者遇到return语句，就会返回，且释放其占用的栈空间，遵循在哪里调用就返回到哪里的原则。

递归实例分析

斐波那契数列

题面：请用递归的方式求出斐波那契数列的第n项。
分析：根据斐波那契数列的递推公式：f(n) = f(n-1) + f(n-2)(n>2)，可初步确定思路。
例程：

public class Maze{
public static void main(String [] args){
int map[][] = new int[8][7];
for(int i = 0; i < 7; i++){
map[0][i] = 1;
map[7][i] = 1;
}
for(int i = 0; i < 8; i++){
map[i][0] = 1;
map[i][6] = 1;
}
map[3][1] = 1;
map[3][2] = 1;

}

}
