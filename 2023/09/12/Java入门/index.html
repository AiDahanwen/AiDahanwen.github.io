Java入门——Java中的数组

1.数组的定义

格式：<数据类型> <数组名> = { } ;

2.Java中数组的使用

  • 动态初始化方法一：
    • 数组的定义：
      格式：<数据类型> <数组名>[] = new <数据类型>[大小]
      举例：int array[] = new int[5];
    • 数组的引用/访问/获取：
      与C语言相同，不再赘述，注意下标从0开始即可。
  • 动态初始化方法二：
    • 先声明数组：
      格式：<数据类型> <数组名>[];或者 <数据类型>[] 数组名;
      示例：int a[]; / int[] a;
    • 然后再创建数组：
      格式： <数组名> = new <数据类型>[大小];
      示例：a = new int[10];
  • 静态初始化：
    • 初始化数组：
      格式：<数据类型> <数组名>[] = {元素值，元素值...};
      示例： int a[] = {1,2,3,4,5,6};

7.二维数组

  • 二维数组在内存中的存储方式
    二维数组名指向存储有每个元素（一维数组）首地址的空间，再由一维数组的首地址访问一维数组中的每个元素。
  • 二维数组的创建
    理解了二维数组在内存之中的储存方式后，对二维数组的声明需要注意的地方就比较好理解了。 必须要申请两遍空间，第一次申请存储每个一维数组的首地址，第二次申请存储每个一维数组的每个元素。
  • 二维数组的遍历,使用了二维数组的元素是一维数组的概念。 即使用array.length即可得到二维数组中有多少个一维数组，使用array[i].length可以得到下标为i的一维数组的元素个数。

8.Java中数组的细节与注意事项

  • 数组是多个相同类型数据（或者能实现数据类型转换的，比如int->double）的集合，实现对数据的统一管理。
  • 数组中的元素可以是任何数据类型，包括基本类型(byte,short,int,long,float,double,char,boolean)和引用类型(class,interface,数组)，但是不能混用
  • 数组创建后，如果没有赋值，则有默认值： int/short/byte/long->0; float/double->0; char->\u0000; boolean->false; String->null.
  • 数组属于引用数据类型，数组型数据实际上是对象(object).

一些零碎的点

  • Java中似乎没有占位符输出这种用法，通常使用较为便捷的字符串拼接代替。
    示例：System.out.println("Your score is " + socre);
  • 在Java中由ASCII码获得字符的方式必须使用强制类型转换
    举例：想在array数组中存储A-Z 26个英文字母时：array[i] = (char)('A' + i);
  • 在求一个数组的最大最小值时，可把储存最值的变量初始化为元素中的任意一个元素。
  • 写冒泡排序的时候务必注意内外层循环的边界！否则就会导致内部array[j + 1]越界，把0交换进数组里出现bug
